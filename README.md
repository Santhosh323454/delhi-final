# AyurSutra - Delhi Hackathon 2026

AyurSutra is an AI-powered Ayurvedic healthcare management system that bridges the gap between practitioners and patients using Gemini AI.

## 🚀 Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Spring Boot, Java, MySQL
- **AI Integration:** Gemini Vision API & AI Voice Services

## 🛠️ How to Run Locally
1. Clone the repository.
2. Setup environment variables in a `.env` file (Refer `.env.example`).
3. Run the Spring Boot backend on port `8080`.
4. Run the React frontend using `npm run dev`.

## ✨ Key Features
- Smart Patient Management
- AI-driven Treatment Planning
- Real-time Analytics Dashboard
